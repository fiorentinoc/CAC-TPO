Proyectos

Core WAN

Con la irrupción de la tecnología MPLS a las prestadoras de servicios de telecomunicaciones, me proponen como PM para implementar un nuevo Core WAN. El proyecto duró dos años y medio para instalar, configurar y migrar equipamiento en 2 datacenters activo-activo y 800 sucursales en todo el país.

Consolas remotas

Con la llegada de la pandemia y las restricciones de movilidad, surgió la necesidad de obtener un acceso rápido a las consolas seriales de los equipamientos. Esto me llevó a construir un sistema que a traves de una Raspberry Pi con un hub USB, conversores USB-TTL y adaptadores de niveles UART, obtener las consolas de forma remota.

Tecnología SDWAN

Actualmente estoy implementando un Core SDWAN de Huawei para conectar 70 sucursales y evaluar los beneficios de esta tecnología en un esquema Hub&Spoke de doble datacenter.

Consulta de APIs

Desarrollo una aplicación de monitoreo web. Consulta una API REST de sistema de Gestion SDWAN y evalúa el estado de salud de una sucursal dada. Basado en Python, con un frontend con tecnología HTML5+CSS3+Bootstrap5+JavaScript
